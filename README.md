# NBA_Record_Website
NBA Taiwan LotterySports Record Website

使用python網路爬蟲 自動化紀錄NBA台灣運彩盤口資料（讓分盤，大小分盤）並自動化更新比賽賽果及分析。
動態產生每日盤口表格與頁面及顏色判斷。
